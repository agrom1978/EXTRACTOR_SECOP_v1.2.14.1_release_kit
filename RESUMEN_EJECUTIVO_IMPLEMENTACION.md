# RESUMEN EJECUTIVO - AutoSECOP1

AutoSECOP1 automatiza la extraccion y consolidacion de datos de procesos SECOP I,
reduciendo tiempo operativo y errores manuales en la captura de informacion.

## Alcance
- UI web para lotes de constancias.
- Extraccion automatizada con navegador controlado.
- Salida estructurada en Excel (plantilla oficial).
- Manejo de errores y reporte por lote.

## Beneficios
- Ahorro de tiempo en captura manual.
- Consistencia en la salida de datos.
- Trazabilidad de errores por constancia.
- Operacion asistida para reCAPTCHA.

## Estado

- Pruebas automatizadas disponibles.
- Documentacion tecnica incluida.
